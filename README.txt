By opening the Assignemnt1AI folder, 
running the main script main.py through the command line "python main.py"
will ask you for the size of maze to load i.e. "Enter 1 for small maze, 2 for medium maze, 3 for large maze: "
and will display the generated maze with the five search algorithms displayed in the order, DFS - BFS - A* - Policy Iteration - Value Iteration
and the metric results will be displayed in the command window

The individual search algorithms can be run for a given maze by following the given examples:
python dfsSearchSmall.py
python dfsSearchMedium.py
python dfsSearchLarge.py

python bfsSearchSmall.py
python bfsSearchMedium.py
python bfsSearchLarge.py

python aStarSearchSmall.py
python aStarSearchMedium.py
python aStarSearchLarge.py

python policyIterationSearchSmall.py
python policyIterationSearchMedium.py
python policyIterationSearchLarge.py

python valueIterationSearchSmall.py
python valueIterationSearchMedium.py
python valueIterationSearchLarge.py
